### 2.2 섹션별 상세

#### 2.2.1 Trade Mode 섹션

| 요소 | ID | 타입 | 사양 |
|------|-----|------|------|
| Export 카드 | `trade-export` | 선택 카드 | 기본 선택 |
| Import 카드 | `trade-import` | 선택 카드 | |

**인터랙션**:
- 카드 클릭 시 `.active` 클래스 토글
- 선택 시 Summary 업데이트

#### 2.2.2 Shipping Type 섹션

| 요소 | ID | 타입 | 사양 |
|------|-----|------|------|
| Ocean | `ship-ocean` | 선택 카드 | 아이콘: 🚢 |
| Air | `ship-air` | 선택 카드 | 아이콘: ✈️ |
| Truck | `ship-truck` | 선택 카드 | 아이콘: 🚛 |

**인터랙션**:
- 선택 시 Load Type 옵션 변경
- Ocean → FCL/LCL
- Air → Air (단일)
- Truck → FTL/LTL

#### 2.2.3 Route & Schedule 섹션

| 요소 | ID | 타입 | Validation |
|------|-----|------|------------|
| POL | `pol-input` | Autocomplete | 필수, Port 형식 |
| POD | `pod-input` | Autocomplete | 필수, Port 형식 |
| ETD Year | `etd-year` | Number (4자리) | 필수 |
| ETD Month | `etd-month` | Number (2자리) | 1-12 |
| ETD Day | `etd-day` | Number (2자리) | 1-31 |

**Autocomplete 동작**:
1. 2자 이상 입력 시 검색 시작
2. 드롭다운에 결과 표시 (최대 20개)
3. 선택 시 전체 문자열 입력
4. 형식: `{CODE} - {NAME}, {COUNTRY}`

#### 2.2.4 Cargo Details 섹션 (FCL)

| 요소 | 타입 | 사양 |
|------|------|------|
| Container Type | Dropdown | 20'GP, 40'GP, 40'HC 등 |
| Quantity | Number | 최소 1, 기본 1 |
| 삭제 버튼 | Icon Button | 행 삭제 (최소 1행 유지) |
| 추가 버튼 | Button | 새 행 추가 |

**동적 행 관리**:
- 최대 10행까지 추가 가능
- 첫 번째 행은 삭제 불가
- 행 번호 자동 표시

## 7. 인터랙션 정의

### 7.1 Autocomplete

```
[상호작용 흐름]

1. 사용자 입력 시작 (2자 이상)
   │
   ▼
2. Debounce (300ms)
   │
   ▼
3. API 호출: GET /api/ports?search={query}
   │
   ▼
4. 드롭다운 표시 (최대 20개)
   │
   ├── 키보드 ↑↓ 탐색
   ├── Enter로 선택
   └── 마우스 클릭 선택
   │
   ▼
5. 입력 필드에 선택값 입력
   │
   ▼
6. 드롭다운 닫기
```

### 7.2 날짜 입력 (분리형)

```
[연도] - [월] - [일]

동작:
1. 각 필드 숫자만 입력 가능
2. 최대 길이 도달 시 다음 필드로 자동 이동
3. blur 시 범위 검증 (월: 1-12, 일: 1-31)
4. 과거 날짜 입력 시 에러 표시
```

### 7.3 모달

```
[열기]
1. 트리거 클릭
2. Overlay 표시 (opacity 0 → 0.5)
3. Modal 표시 (scale 0.9 → 1, opacity 0 → 1)
4. Body scroll 비활성화

[닫기]
1. X 버튼 / Overlay 클릭 / ESC 키
2. Modal 숨김 (scale 1 → 0.9, opacity 1 → 0)
3. Overlay 숨김
4. Body scroll 활성화
```

### 7.4 Toast 알림

```
┌─────────────────────────────────────────┐
│  ✅ 입찰이 성공적으로 제출되었습니다.    │
└─────────────────────────────────────────┘
```

| 타입 | 아이콘 | 배경색 |
|------|--------|--------|
| Success | ✅ | #22c55e |
| Error | ❌ | #dc2626 |
| Warning | ⚠️ | #eab308 |
| Info | ℹ️ | #3b82f6 |

**동작**:
- 우측 상단에 표시
- 3초 후 자동 사라짐
- Fade out 애니메이션

### 7.5 로딩 상태

| 상황 | 표시 |
|------|------|
| **전체 페이지** | 전체 화면 스피너 + 반투명 Overlay |
| **섹션** | 섹션 내 스피너 |
| **버튼** | 버튼 내 스피너 + 텍스트 변경 |
| **카드** | 스켈레톤 UI |
